In this chapter, we looked at Javascript, more often abbreviated as JS, and how this important component of webpage design. Javascript is a high-level, interpreted programming language, and it serves as one of the technologies of World Wide Web content engineering, making webpages interactive and providing online programs, including video games. One common misconception between the general public regarding Javascript is regarding this programming language as the same as Java, but aside of having a lot of similities, including language name, syntax, and respective standard libraries, the two languages are distinct and differ greatly in design as well as their intended uses. For starters, Javascript is used to add client-side behavior to HTML pages, as well as being capable of including multimedia and graphics.
